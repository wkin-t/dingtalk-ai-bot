# Security Policy

## Supported Versions

This project currently supports security fixes for the `master` branch only.

## Reporting a Vulnerability

Please report vulnerabilities through GitHub private vulnerability reporting:

- Go to the repository's **Security** tab.
- Click **Report a vulnerability**.
- Provide reproduction steps, impact, and suggested remediation.

Do not disclose vulnerabilities in public issues before a fix is available.

## Response Targets

- Initial triage response: within 72 hours
- Status updates: at least every 7 days while a fix is in progress
- Coordinated disclosure: after patch release and mitigation guidance is published
